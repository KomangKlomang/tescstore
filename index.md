# Halo Dunia! 👋

Selamat datang di website saya! Ini adalah bukti bahwa Anda bisa membuat website yang di-host di **GitHub Pages** hanya dengan bermodalkan 1 file `index.md` saja.

## Mengapa ini bisa terjadi?
GitHub Pages secara otomatis menggunakan Jekyll di belakang layar. Meskipun kita tidak membuat file konfigurasi (`_config.yml`), GitHub akan menggunakan pengaturan defaultnya untuk mengubah file Markdown ini menjadi halaman HTML yang cantik.

### Daftar Kelebihan:
* **Sangat Mudah:** Tidak perlu banyak file.
* **Cepat:** Tinggal tulis Markdown, langsung jadi website.
* **Gratis:** Hosting gratis selamanya dari GitHub.
